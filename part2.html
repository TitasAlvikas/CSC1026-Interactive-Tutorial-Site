CSS introduction

Cascading Style Sheets were first introduced in 1994 alongside the development of HTML and give separation of webpage content and presentation. When HTML was developed it was never intended to handle the presentation of webpages and so the introduction of CSS was able to solve a very large problem in web development.

CSS syntax

CSS rules are defined inside curly braces in the form: style-property: value; External style sheets can be linked to a web document using the <link> tag:

<link rel="stylesheet" type="text/css" href="cssfileURL.css" />

HTML elements can be styled by the tag name. Class attributes are used as selectors to style certain HTML elements.

Id attributes are used as selectors for unique elements.

Advanced CSS properties

CSS units can be specified as a measurement (absolute units) of either pixels (px), points (pt) or millimetres (mm). CSS units can also be specified relative to other page elements (relative units) as percentages (%), emes (em) or exes (ex)

There is more than one way to specify a colour for page elements. The available options are:
color: blue;
color: rgb(0,67,235);
color: AF385F;

CSS has some pseudo elements aimed at styling a special part of a tag. tagname:first-letter can select the first letter of body of text.

Additional pseudo classes can change the styling of an element based on a special status of some elements.

CSS box model

Fundamental part of the box model is that each element on the page is represented as a box.

The border of a box is the outline of the box represented by three separate properties. These are the border-width, border-style and border-color and has the following shorthand notation. Note: CSS3 provides additional supprt for rounder box borders using the border-radius property and shadows around borders using the box-shadow property.

A padding is the space between the outside edge of the content and the inside edge of the box. The space created by the padding is always inside the box.

The margin of the CSS box model is the space around the outside edge of a box including the box border.

The width and height properties specify the dimensions of only the content width and height of a box. The min-width and max-width properties can also be set to specify the minimum and maximum sizes of a box. Note: IE box model uses CSS width for the total dimensions of a box. IE+ now uses the standard model if a correct DTD has been declared.

CSS positioning

CSS boxes can be floated to the side of a webpage giving us a very good way of positioning some content. It is worth noting that when an element is floated it is taken out of the natural page flow and the next element takes up the background space of the floated box.

The clear property can be used to stop a float from covering later content under a box. The clear property can take the values left, right or both to specify which side of a box to clear any floats.

The position property allows you to specify exact positions from certain elements. The relative value can position an element relative to its original position in the natural page flow. You can specify the distances from the top left of an element's original positiom using the left, right, top and bottom properties. Note: The relative position will still leaves the element in the page flow.

The absolute position takes an element out of the page flow and positions it with the specified offset from it's parent container.

Fixed positioning takes an element from the page flow and fixes it to the browser viewing screen so that scrolling will not affect the element's position.
